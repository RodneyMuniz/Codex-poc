# Architecture Decision

- chosen_approach: run one explicit supervised operator CLI rehearsal in an isolated workspace
- rationale: prove sanctioned packet/bundle, inspection, and gate flow without broadening authority
- tradeoffs: later-stage execution, apply/promotion, and unattended behavior remain out of scope
- dependencies: first-slice stage artifacts, provider proof, pending bundle review
- downstream_implications: next work should harden end-to-end CLI invocation and apply/promotion under supervision
